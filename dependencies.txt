# Database

First time - docker run -d --name intern-mongo -p 27017:27017 mongo:6

Next time  - docker start intern-mongo

Access database - docker exec -it intern-mongo mongosh

.env file contents
PORT=4000
MONGO_URI=mongodb://localhost:27017/intern_assignment
JWT_SECRET=supersecretkey
JWT_EXPIRES_IN=15m
CORS_ORIGIN=http://localhost:5173
RATE_LIMIT_WINDOW_MS=60000
RATE_LIMIT_MAX=100
